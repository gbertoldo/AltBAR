# AltBAR-UI 🚀 
 Interface gráfica para altímetros AltBAR.

# Versão
Versão 1.0.0

# Como usar
1. [Baixe](https://github.com/gbertoldo/rRocket-UI/releases) a versão mais recente do executável (disponível apenas para Windows).
1. Descompacte o arquivo zip baixado.
1. No diretório do arquivo extraído há o executável AltBAR-UI.exe. Não é necessário realizar instalações, basta rodar esse arquivo. Se não houver erros, surgirá uma janela como ilustrado na figura abaixo:
